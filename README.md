### Energy-Efficient LPWAN Smart City Monitoring Framework

<a href="https://github.com/ahmed-sh003/Smart-City-LoRaWAN">

</a>

**Graduation Project | Team Leader | AIoT | LPWAN | Smart Cities**

An end-to-end smart infrastructure monitoring framework using **ESP32-based LoRa sensing nodes**, **Firebase Realtime Database**, **Flutter dashboards**, **OMNeT++ / LoRaSim simulation**, and **AI-assisted recommendation models**.

#### Key Highlights

- Built smart infrastructure monitoring for:
  - **Building & Irrigation**
  - **Bridge & Road**
  - **Water Network**
- Implemented long-range communication using **ESP32 + SX1278/SX1276 LoRa modules**.
- Integrated gateway communication with **Firebase Realtime Database**.
- Designed mobile dashboards with **Admin** and **Technician** workflows.
- Validated approximately **650 m outdoor communication range**.
- Achieved node-level **PDR from 66.7% to 86.1%**.
- Uploaded **230 successful cloud records** with no observed CRC or packet-format errors.
- Developed AI recommendation models using node telemetry data, achieving **81% and 83% accuracy**.
- Worked on dual-band LoRa optimization using **433 MHz** and **868 MHz**.

**Tech Stack**

### Energy–Reliability Trade-offs in Dense LoRaWAN

**Status:** Peer Review Round Passed  
**Journal:** *Signal, Image and Video Processing, Springer*

A simulation-based LoRaWAN study evaluating **channel diversity**, **SINR-aware adaptive retransmission**, **energy consumption**, and **battery lifetime** under dense urban deployment scenarios.

**Reported Results**

- Increasing uplink channels from **1 to 3** improved PRR from **0.9352 to 0.9890**
- Reduced daily energy consumption by **21.8%**
- Extended battery lifetime by **29.1%**
- Adaptive retransmission reduced retransmission pressure by **45.6%**
- Improved battery lifetime by **9.4%** at **400 nodes**
